## Current Project

### Community Web Platform

React + Spring Boot 기반의 포털형 웹 플랫폼을 개발하고 있습니다.  
커뮤니티, 검색, 블로그, 할 일 관리, 회원 기능을 하나의 서비스 흐름으로 연결하는 것을 목표로 하고 있습니다.

| Area | What I Built |
|---|---|
| Auth | 회원가입, 로그인, 아이디 찾기, 비밀번호 재설정, 닉네임 변경 |
| OAuth2 | Google/Kakao 로그인 연동 구조 |
| Community | 게시판, 게시글 CRUD, 댓글 CRUD, 사용자별 게시글 조회 |
| Search | Naver 검색 API 기반 웹/뉴스/블로그/이미지 검색 |
| AI | Gemini API 기반 검색 요약 기능 |
| Productivity | Todo 생성, 수정, 상태 변경, 삭제 |
| Career | WorkNet API 기반 채용 정보 조회 |
| Backend Structure | Controller, Service, Repository, DTO, Entity 계층 분리 |

## Project Direction

단순히 화면을 따라 만드는 포트폴리오보다,
