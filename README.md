## Projects

### [Flatform](https://github.com/breadcream177/flatform)

React + Spring Boot 기반의 개인 웹 애플리케이션입니다.  
커뮤니티, 검색, Todo, 블로그, 회원 기능을 포함하고 있으며, 프론트엔드와 백엔드를 분리한 구조로 개발했습니다.

- Frontend: React, TypeScript, Vite
- Backend: Java, Spring Boot, Spring Data JPA, Spring Security
- Database: MariaDB
- 주요 기능: 회원가입/로그인, 게시판, 댓글, Todo, 블로그, 외부 API 검색, AI 요약
- 정리 내용: ERD, API 명세, 실행 방법, 프로젝트 구조

---

### [Pet Project](https://github.com/breadcream177/pet-project)

반려동물의 밥, 산책, 약, 병원 일정을 관리하는 모바일 중심 웹/앱 프로젝트입니다.  
Next.js와 Supabase를 기반으로 사용자별 데이터 분리, 일정 관리, 완료 체크, 알림 구조를 구현했습니다.

- Frontend: Next.js, React, TypeScript
- Database/Auth: Supabase Auth, Supabase PostgreSQL
- Mobile: Capacitor Android
- Push: Firebase Cloud Messaging
- 주요 기능: 반려동물 등록, 일정 등록, 반복 일정, 오늘 일정 체크, 완료 기록 저장, 알림 토큰 저장

---

### [Electronics Shop](https://github.com/breadcream177/electronics-shop)

PHP와 MySQL로 구현한 전자제품 쇼핑몰 개인 프로젝트입니다.  
REST API 서버 방식이 아니라 PHP 페이지와 Form 요청을 중심으로 동작하는 서버 렌더링 방식으로 구현했습니다.

- Backend: PHP
- Database: MySQL / mysqli
- Frontend: HTML, CSS, JavaScript
- Auth: PHP Session, 일반 로그인, 카카오 로그인, 구글 로그인
- Payment: PortOne/Iamport, KakaoPay
- 주요 기능: 상품 목록/상세, 검색/필터/정렬, 장바구니, 위시리스트, 주문, 결제 검증, 리뷰, Q&A, 관리자 주문 관리
